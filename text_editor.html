SOBRE O PROJETO

Um editor de texto escrito em C/C++.

Escolhi a escrita de um editor de texto para ser meu primeiro "grande" projeto pois é desafiador o bastante para meu nível de conhecimento e me mantém motivado, pois nele encontro diversos assuntos pelo que tenho interesse.

O primeiro renderizador foi escrito em software por motivos de aprendizado, mas no futuro vou portá-lo para um renderizador comum acelerado em hardware.

Para o carregamento das bitmaps das fontes, foi usada a biblioteca "stb_truetype". Todo o resto do editor foi escrito por mim apenas utilizando as apis WIN32 do Windows e a bibliotecas padrão da linguagem C/C++.

A linguagem escolhida é um dialeto de C++ que é basicamente C com algumas adições de C++ como default parameters, declare anywhere, algo que se assemelha bastante ao C++ que é usado na documentação da api Win32.

Como inspiração gráfica para o design do editor nesta primeira iteração, usei como bases editores como EMACS, 4Coder, Vim, que simplificam o design UI e usam mais o teclado para manipular o editor ao invés do mouse, porém planejo adicionar suporte ao mouse e oferecer uma UI "clicável" como outros editores.
